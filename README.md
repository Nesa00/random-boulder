# Boulder Randomizer

A simple web-based randomizer for bouldering problems.

This project allows climbers to randomly select a boulder problem between a chosen minimum and maximum grade. It also keeps a history of previous selections and provides a way to reset that history.

## Features

- Choose a minimum grade
- Choose a maximum grade
- Pick a random problem
- View previously selected problems
- Reset history
